Sobre a tag 'nav'

É usada para agrupar e referenciar links importantes na página, não precisa ser usada em todos os link, mas nos links de navegação principal pode ser usada! Apenas para os grupos de links mais importantes

Dicas sobre o 'nav':

É comum enolver links de navegação com 'ul'(unordered list - lista desordenada) ou 'ol'(ordered list - lista ordenada) juntamente com 'li'(list), então a tag 'nav' não substitui essa função, mas adiciona acessibilidade para informar que aquele é uma lista de links importantes até para navegação pelo site, então devemos envolver estas listas de links em um novo elemento 'nav'

  1. exemplo de uma lista ordenada
  2. próximo item
  • exemplo de lista desordenada
  • próximo item da lista desordenada
