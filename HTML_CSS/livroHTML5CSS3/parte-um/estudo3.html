Sobre TAGs, Atributos, Elementos e Valores:

Falando sobre tags itens fundamentais em HTML, há elementos, há atributos e há valores. Os elementos descrevem o que são aquelas tags e seus significados, por exemplo: Isto aqui é um título, aquilo é um parágrafo... ou seja, são como rótulos que descrevem as diferentes partes de uma página web.

Uma tag (podemos traduzir como ETIQUETA) é o item que representa algo, por exemplo: P /P são tags, pois elas representam que aquilo vai ser um paragrafo. Já elementos são todo o conteúdo junto das tags, por exemplo: P -conteúdo dentro da tag p /P, tudo isso é considerado um elemento que descreve um paragrafo.

Os elementos podem conter algo (texto) ou serem vazios. Os elementos não vazios são as tags de abertura, o conteúdo e as tags de fechamento com uma barra final. Um elemento vazio é a combinação das tags de abertura e fechamento em uma única tag, por exemplo: CHAVES img BARRA CHAVES. Há alguns que acabam não usando as barras de fechamento nesses elementos com o HTML 5 , mas acho bom usá-las.

Então como nas imagens, um elemento é tudo que envolve as tags de abertura e fechamento e seus atributos, valores e conteúdos, por exemplo: uma tag P com texto dentro e fechamento da tag P, tudo isso é um elemento de parágrafo!

Normalmente é usado caixa baixa para descrever as tags. Os elementos vazios não rodeiam nenhum conteúdo, mas podem ter atributos e valores assim como os elementos não vazios. Ou seja, podemos dizer que as estruturas dos elementos html envolvem: tags de abertura e fechamento dependendo do tipo de elemento a ser usado, atributos e seus valores, e o conteúdo envolto pelas tags se for elemento não vazio!

Sobre atributos:

Os atributos contêm informações sobre os conteúdos daqueles elementos, tanto elementos vazios quanto elementos não vazios. A ordem dos atributos não importa desde que sejam separados por um espaço. Há atributos que podem ser atribuidos valores diversos, mas há os atributos que somente vão aceitar valores pré-definidos!

- Aqui temos um elemento do tipo label que possui um atributo de nome FOR e ele aceita nomes, então ele possui a valor de nome Email. Já um atributo REL aceita valores pré-definidos como STYLESHEET por exemplo.

Há também atributos como o SRC e o HREF que somente aceitam valores na forma de uma URL!
