Box Model:

Content = CONTEÚDO: Define a largura inicial da caixa(salvo elementos do bloco).

Padding = PREENCHIMENTO: Separa o contúdo das bordas da caixa. É a margem interna.

Border = BORDA: Define bordas da caixa.

Margin = MARGEM: Define a distância entre uma caixa e outra.

Width = LARGURA: A largura total da caixa, por padrão é o somatório do conteúdo + padding(left/right) + border(left/right).

Height = ALTURA: A largura total da caixa, por padrão é o somatório do conteúdo ++ padding(top/bottom) + border(top/bottom).


Pixel: É a unidade de referência da Web, pois as telas são desenvolvidas em pixels.

Monitores: um monitor por exemplo, 3840(largura em px) x 2160(altura em px) = 8.294.400, Monitor 4K

px em CSS: No CSS o pixel (px) é uma unidade de referência e não representa 1 pixel exato do seu dispositivo!!! (é adaptável em relação á densidade da tela).

div: A DIV é um elemento de bloco, block genérico que serve para auiliar nos posicionamentos dos elementos/conteúdos na tela. Exitem elementos semânticos como: MAIN, SECTION, NAV...
