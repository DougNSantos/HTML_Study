Acesse meus cursos neste link: este link tem um padding de 20px agora sendo respeitado obs: a tag A é um elemento inline, por isso ele acaba transbordando para outros elementos dependendo da definição que fazemos. pode passar por cima de outro elemento de bloco por exemplo
usando a definição no css - display: inline-block, ele vai respeitar o bloco, então o elemento pai acaba tendo o valor do conteúdo que está usando

DISPLAY: INLINE-BLOCK - o elemento continua inline, mas passa a receber as propriedades do box model, como padding, margin... , assim o conteúdo passa a respeitar e aplicar esses efeitos, sem esse display, o elemento em linha pode mostrar a borda sendo definida no css por exemplo, mas não estará sendo aplicado a ele essa propriedade!

a propriedade - display: none; - remove o elemento completamente da tela
