Existem propriedades no css que são passados de pai para filho, então se aplicarmos um estilo no //body, quase todos elementos que estiverem
dentro do body serão afetados com aquele estilo. Alguns elementos tem a pré-definição de estilo, então normalmente aqueles não serão afetados
mas podemos aplicar ou remover aqueles estilos
