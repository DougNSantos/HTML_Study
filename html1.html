Uma página WEB é basicamente formada por 3 elementos HTML, sem HTML, por mais básico e simples que seja, não conseguimos
criar uma página WEB: CONTEÚDO DE TEXTO, texto que aparece na página, informa seus negócios
REFERÊNCIAS A OUTROS ARQUIVOS: carregam itens como imagens, audio, vídeo e arquivos SVG(scalable vector graphics)
MARCADORES: elementos HTML que descrevem o conteúdo de seu texto e fazem as referências funcionarem.
APRIMORAMENTO PROGRESSIVO: crido por Steve Champeon, é uma boa prática para criar páginas web acessíveis a tudo e a todos,
consiste em criar a estrutura básica do site em HTML e após ir melhorar com CSS e JavaScript, assim navegadores que não
conseguem usar CSS por exemplo, mesmo assim mostrem as informações ao usuário, e aqueles navegadores mais modernos, mostrem a
mesma página com visual com CSS.
A marcação(markup) DESCREVE o significado do conteúdo, mas não aparece ao usuário final, apenas no código HTML.
Cada linha no código tem uma certa quebra de linha, para funcionar não é necessário, mas isso se chama Identação e
é muito importante para manutenção e escrita de código, é uma boa prática.

Tudo acima da tag Body, é informção instrucional para os navegadores e não é apresentado ao usuário, apenas a tag Title é mostrada
Importante é sempre começar com código HTML informando DOCTYPE, pois isso informa ao navegador qual versão do HTML estamos usando!!!
O doctype do HTML5 é !DOCTYPE html, com maior e menor (braquets) entre eles.

As tags HTML, indicam o começo e o fim de um documento html. No head, vão as informações do site para o navegador, title é visível.
No body, vão o corpo da página, ou seja, seu conteúdo.
