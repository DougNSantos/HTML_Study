O essencial de uma página HTML 5:

!DOCTYPE html- informa ao navegador qual versão estamos usando de html, nesse caso é o HTML5
html- inicia um documento html, junto dele podemos informar a linguagem que estamos escrevendo o documento, ex= html lang="PT-br"
assim como o fechamento de todo documento html se encerra em /html
head- vai toda informação do documento para o navegador, poucas informações aparecem ao usuário exemplo: title nos mostra o título da página
, já as tags meta..... informam dados da página para o navegador processar
body- aqui vai tudo que aparece na página, textos, links, dados de vídeo,audio,imagem......
Depois do básico e essencial, nós fechamos as tags abertas de /body, /html(finaliza o documento html) e as tags anteriores devem ser fechadas
em suas respectivas posições, como a tag /head.
